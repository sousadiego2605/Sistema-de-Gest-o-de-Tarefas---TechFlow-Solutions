# Sistema-de-Gest-o-de-Tarefas---TechFlow-Solutions
Gerenciador de Tarefas TechFlow Sistema ágil para uma startup de logística focado em monitoramento em tempo real e priorização de tarefas. Desenvolvido com foco em Engenharia de Software, utiliza GitHub Projects para gestão via Kanban e GitHub Actions para CI/CD com testes automatizados, garantindo qualidade e entrega contínua.
